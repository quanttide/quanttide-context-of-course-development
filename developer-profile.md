## 身份

独立开发者/小团队 lead，在 building **量潮课程云（qtcloud-course）**。

## 工作特征

- **产品 + 基础设施并行推进**：一边交付课程云产品（Flutter 桌面客户端、课程蓝图、视频录制计划），一边搭建工程基础设施（GUI 自动化测试框架、qtcloud-devops 发布流水线、submodule 编排、contract.yaml 组件化管理）
- **抽象分层意识强**：领域模型有 Program → Course → Phase → Lesson → Sense → Step 六级结构，对应不同视频长度和交付粒度。代码同样体现分层思想（工具库与易变逻辑分离）
- **重视知识闭环**：会把自己踩过的坑整理成可传递文档（如 GUI 自动化测试方法、从二进制逆向提取功能文档）
- **多语言多技术栈**：Python（测试）、Flutter/Dart（桌面端）、Rust（CLI）、Go（provider）、系统脚本（xvfb/xdotool/ImageMagick/tesseract），各语言用在最合适的地方
- **决策链路短，行动力强**：对效率敏感，但该建的基础设施一样不少，知道什么时候快、什么时候稳
